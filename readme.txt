Let's you set an image used for social Sharing (such as Facebook and Twitter) for you homepage / blog page. It does NOT do anything for single posts and pages as those are usually already handled via featured images and the theme or other SEO plugins. 

== Description ==

Plugin that displays Facebook's Open Graph image tag on the homepage with an image of choice. It does not touch any other pages. It works on the static homepage and on the "blog" page that shows the latest posts.

== Installation ==

1. Upload `simple-homepage-facebook-og-image` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set up a the image via Settings -> Simple Homepage Facebook Imag

== Frequently Asked Questions ==
This plugin is ONLY for setting the featured image, title and description for the homepage and blog page.
